# PROJECT GOAL

Pi-GATO is a live-performance video playback instrument designed for:

* instant clip triggering
* persistent video engine playback
* low-latency numpad control
* live bank switching
* timestamp saving
* safety clip fallback
* modular performance modes

The architecture evolved from:
simple mpv playback scripts

into:

a persistent IPC-driven live media engine.
